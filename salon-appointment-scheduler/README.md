# 🦖 Dino-Spa Appointment Scheduler
A relational database and bash script project built for the [FreeCodeCamp Relational Database Certification](https://www.freecodecamp.org/learn/relational-databases-v9/). This application manages a prehistoric salon where dinosaurs can book services like "The Meteor Buffer" or a "Pterodactyl Wax" using a terminal-based interface. 


## 🛠️ Technologies Used

*   **PostgreSQL**: Relational database for storing customers, services, and appointments.
*   **Bash**: Shell scripting for the interactive user interface and database communication.
*   **SQL**: Database schema design and data manipulation.
